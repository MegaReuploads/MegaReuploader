# MegaReuploader

MegaReuploader (MR) é um programa de código aberto capaz de "arquivar" um vídeo, uma playlist, ou até mesmo um canal inteiro do YouTube, com o simples uso da URL, graças à programação Python e a biblioteca yt-dlp, e utilizando o FFMPEG.
Entrando mais a fundo sobre "arquivar", o programa dá a opção ao usuário de baixar as thumbnails do vídeo, metadados (likes, descrição, título, views, etc.) e escolher a qualidade do vídeo, tudo diretamente no terminal do Windows.

O MR é capaz de sustentar uma comunidade de reuploaders de vídeos apagados e de lost medias do YouTube, sendo capaz de arquivar vídeos que até mesmo o WayBack Machine não consegue.

Em geral, o MR é um programa com uma UI simples, e amigável ao usuário, onde qualquer um pode usar para baixar seus vídeos preferidos, ou arquivá-los antes que sejam excluídos para sempre.

## Quem somos?

Somos MegaReuploads, uma equipe sem fins lucrativos cujo objetivo é arquivar vídeos de diferentes idiomas do YouTube.
No nosso [canal do YouTube](https://www.youtube.com/@MegaReuploads), pode encontrar qualquer tipo de vídeo de diversos canais, tanto de canais excluídos quanto de vídeos não listados, privados ou excluídos.

Nós fazíamos a arquivação dos vídeos manualmente, indo atrás das thumbnails, descrições e metadados dos vídeos, um por um. Vimos o quão demorado isso podia ser, e resolvemos criar um programa capaz de fazer essa arquivação, de forma eficiente.

Após o sucesso do programa em fazer tais arquivações, resolvemos polir ele, e transformá-lo publico para todos usarem, e contribuírem também, salvando e arquivando os vídeos.
