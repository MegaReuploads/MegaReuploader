# MegaReuploader
[Portuguese]
MegaReuploader (MR) é um programa de código aberto capaz de "arquivar" um vídeo, uma playlist, ou até mesmo um canal inteiro do YouTube, com o simples uso da URL, graças à programação Python e a biblioteca yt-dlp.
Entrando mais a fundo sobre "arquivar", o programa dá a opção ao usuário de baixar as thumbnails do vídeo, metadados (likes, descrição, título, views, etc.) e escolher a qualidade do vídeo, tudo diretamente no terminal do Windows.

Em geral, o MR é um programa com uma UI simples, e amigavel ao usuário, onde qualquer um pode usar para baixar seus vídeos preferidos, ou arquivalos antes que sejam excluidos para sempre.

O MR é capaz de sustentar uma comunidade de reuploaders de vídeos apagados e de lost medias do YouTube, sendo capaz de arquivar vídeos que até mesmo o WayBack Machine não consegue.


[English]
MegaReuploader (MR) is an open-source program capable of "archiving" a video, a playlist, or even an entire YouTube channel, simply by using the URL, thanks to Python programming and yt-dlp.
Going deeper into "archiving," the program gives the user the option to download video thumbnails, metadata (likes, description, title, views, etc.) and choose the video quality, all directly in the Windows terminal.

MR is able to support a community of reuploaders of deleted videos and lost media from YouTube, being able to archive videos that even the Wayback Machine cannot.

In general, MR is a program with a simple, user-friendly UI that anyone can use to download their favorite videos or archive them before they are permanently deleted.
